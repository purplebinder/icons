# Purplebinder::Icons

Webfont binder icons, like FontAwesome. We host this on RubyGems for ease of access.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'purplebinder-icons'
```

## Usage

    // application.css
    *= purplebinder-icons
    
    <!-- In your markup -->
    <i class="pb pb-binder"></i>
